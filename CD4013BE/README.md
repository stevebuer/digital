# TI CD4013BE Dual D-Type Flip-Flop

Texas Instruments CD4013BE dual flip-flop in a 14 pin DIP.

[Data Sheet](https://www.ti.com/lit/ds/symlink/cd4013b.pdf?ts=1752127089880)

## Pins

 1: Q
 2: Q Bar
 3: Clock 1
 4: Reset 1
 7: VSS (ground)<br>
14: VDD (power)
